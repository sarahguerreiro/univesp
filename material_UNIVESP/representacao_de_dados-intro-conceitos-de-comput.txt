// Codificação Binária_

Combinações de bits permitem armazenar um numero maior de estados.

/Byte - BinarY TErm: Unidade básica de amazenamento em memória e disco usada pelos
computadores e formada por 8 bits.

/ Palavras:
 Referem-se a um grupo de bits tratados como uma unidade unica pelo processador.
O tamanho da palavra é uma característica importante da arquitetura do pc e afeta diretamente
diversos aspectos do seu funcionamento. A relação entre o tamanho da palavra e a unidade de
transferencia entre a cpu e a memoria principal é fundamental para entender como o processador
interage com a memoria. 

>>> POR EXEMPLO <<<<<
DIgamos que temos um computador com uma arquitetura de 32 bits. Isso significa que o processador
opera com palavras de 32 bits. Cada palavra de 32 bits pode representar um numero na faixa
de 0 a 2^32-1(~4 milhões). Além disso, o tamanho da palavra tbm define a largura 
do barramentode dados - que é a unidade  de transferencia entre a CPU e a memoria principal.

Nesse exemplo de arquitetura de 32 bits, o barramento de arquitetura tbm terá 32 bits de
largura. Isso significa que cada vez que a CPU preisar ler ou escrever dados na memória
principal, ela pode transferir uma palavra de 32 bits de uma só vez. Se a CPU precisar
acessar um dado que ocupa mais de 32 bits na memoria, ela pode precisar realizar varias 
transferencias de dados.

Por outro lado, se tivessemos uma arquitetura de 64bits, onde a palavra tem 64 bits
de tamanho, o barramento de dados tbm seria de 64 bits de largura. Isso permitiria 
transferencias de dados mais eficientes entre a CPU e a memoria principal, ja que 
cada transferencia pooderia conter o dobro de dados em comparação com uma arquitetura
de 32 bits.

Portanto, a relação entre o tamanho da palavra e a unidade de transferencia entre a CPU
e a memoria principal é que o tamanho da palavra determina a largura do barramento de d
ados, influenciando assim na eficiencia das transferencias de dados e o desempenho geral
do sistema.

// CODIFICAÇÃO BINÁRIA_IMAGEM_

Imagens tbm sao codificadas. As imagens sao transformadas em uma matriz 
bidimensional de numeros, onde a unidade básica é um pixel.

/ PIXEL: Menor componente de uma imagem digital.
/ BITMAP: Coleção de pixels de uma imagem. 

O numero de pixels/ unidade de area, determina a resolução de uma imagem.
O pixell é representado geralemnte pelo grau de participação de cada cor basica - azul
verde e vermelho - na cor final da imagem colorida. Codificação RGB(Red, Green, Blue).

/ A extensão bitmap é um outro formato mais direto de representação gráficas, pois 
não sofre nenhum tipo de compactação. Por isso os arquivos, apesar de simples, costumam
ser pesados/grandes.

/Arquivo JPEG: Nesse formato apenas o que é relevante na imagem é usado pra um
melhor aproveitamento do espaco disponivel p armazenamento, ou seja,
sao imagens compactadas.

/ Arquivo Gif: Limita o numero de cores a 256. cada imagem gif pode ser 
composta por diferentes conjuntos de 256 cores.
